# myPieChart
円グラフを使ったスケジュール管理的なもの

## 計画
1. 定数の円グラフを表示（Chart.js）
2. Vue.jsで数値をリアルタイムに反映するようにする。
3. スタイル整える
4. 完 

## 要件
* 円グラフ  
  * Chart.js
  * 円グラフ以外のもの（優先度：低）
* Vue.js
  * cliは使わない。
  * 数値を変更すると円グラフが同期される。
  * 要素を増減できる。
* スタイル
  * 円グラフの色も入力で変更できるようにしたい（優先度：低）
* 技術
  * バージョンアップしたGulpセットを使う。
    * Pug / Scss / Webpack / browser-sync
